## Design Context

### Users

一二线城市年轻养宠人（20–35 岁，以城市公寓生活的猫狗主人为主）。主要使用场景：

- 日间通勤间隙、午休快速刷一眼新帖
- 晚上躺在床上长时间翻阅晒娃、经验帖
- 想把不再用的猫砂盆 / 猫窝 / 粮 出二手，也会搜索本地有无可信的出售者
- 偶尔需要紧急求助（行为问题、饲养问答），期望被温柔、具体地回答

核心 job-to-be-done：

1. **社交分享**：发布、浏览、评论宠物日常（"晒娃"）
2. **经验交换**：求助问答、线下活动、知识沉淀
3. **可信交易**：二手闲置、产品、服务、领养（低风险、信息充分）
4. **私信 / 通知**：跟成交相关的一对一沟通

### Brand Personality

三个词：**温暖、克制、可信**。

- 温暖 ≠ 幼稚。不用卡通喵咪、不用粉色糖果色。
- 克制 ≠ 冷淡。保留暖奶油、暖近黑、暖红的签名色。
- 可信 = 订单、支付、账号安全这类场景必须简洁、无歧义、无表情包。

情绪目标：打开时是"像翻开一本排版讲究的宠物生活杂志 + 顺手又好用的工具"，不是"又一个塞满卡片的管理后台"。

### Aesthetic Direction

**Editorial bold within warm minimalism**——在现有 Cursor 风暖极简基础上把"杂志感"真正落地。

视觉锚点（single source of truth 是 `web/theme.css`）：

- 画布：warm cream `#f2f1ed`（light）/ warm dark `#1b1a16`（dark）
- 正文：warm near-black `#26251e`（不用纯黑）
- 品牌强调：Cursor orange `#f54e00`（链接 / 高亮）
- 签名交互：warm crimson `#cf2d56`（按钮 hover、错误、强反馈）
- 成功：`#1f8a65`（绿）；借鉴 Cursor Agent 的 pastel timeline（think/grep/read/edit）
- 字体三分工：display = Inter（近 CursorGothic）；serif = EB Garamond（近 jjannon，开 `cswh`）；mono = JetBrains Mono（近 berkeleyMono）

**明确的参考 / 反参考：**

- 参考：Cursor 官网、The Browser Company 文档、Are.na、经典印刷杂志的开篇页
- 反参考（绝不要）：通用 SaaS dashboard（蓝灰 + 卡片堆叠）、宠物 App 常见的糖果色 + 大卡通头、AI 生成的 cyan-on-dark 故障风

**明确禁忌（与 impeccable absolute_bans 对齐）：**

- 不用 `border-left: Npx solid color` 当侧条强调（AI-slop 经典）
- 不用渐变文字（`background-clip: text` + gradient）
- 不用纯黑 `#000` / 纯白 `#fff`
- 不用 Inter/Fraunces/Playfair/Space Grotesk/IBM Plex/Outfit 等 reflex_fonts_to_reject；**本项目 Inter 是 Cursor 指定 display 替代，保留；其余 reflex 字体一律不引入**
- 不把每个 h2 上面都挂一个圆角大图标
- 不把整页 3 张一模一样的 stat-card 横排

### Design Principles

1. **Magazine over dashboard.** 首页与内容页的节奏按杂志排，不按数据看板排；标题大胆、正文讲究、间距有呼吸。
2. **One warm red moment per surface.** warm crimson `#cf2d56` 是签名色，每屏保留一次它的瞬间即可（主按钮 hover、链接高亮），滥用立刻失效。
3. **Serif carries story, sans carries action.** 长阅读（帖子正文、引言 lede、说明、评论）走 serif + `cswh`；按钮、表单、导航、标题体走 display sans；订单号 / 代码 / 标签走 mono。
4. **Asymmetry earns trust.** 左对齐 + 主次不等列宽（7/5、8/4）比"两张等宽卡"看起来更像被认真设计过。破格要有意图，不是随机。
5. **Theme follows context, not default.** light 面向日间长阅读（首页 / 内容），dark 面向弱光夜刷（可选），warm 面向偏印刷口味的阅读主题——切换由用户选择，切换字体 / 语言偏好不随主题丢失。
